- Python 3.10
- HTML 5
- CSS 3

### List of libraries:
- PyQT5: Qtcore, QtGUI, QtWidgets, QtWebEngineWidgets
- Requests
- OS
- SYS
- Pathlib
- Time
- Datetime
- Altgraph
- FBS
- Future
- Macholib
- Pefile
- CTypes

```shell
pip install -r requirements.txt
fbs run
```
